1. Extract to C:/ so C:/EmulatorOvelay/ exists
2. Run INSTALL/Install.bat
3. Move ROMs to into respective /ROMS/ folder e.g. NFSU2.iso -> ROMS/PS2Roms
4. Launch with Shortcut created under /Install/ folder, move to desktop if needed. OR use launcher.bat
Extras:
Enable Launch from XBOX Controller: You can move config/XMBEmulator_detect_home_button.bat to 'shell:common startup' to make the emulator overlay accessible from the XBOX Home button on startup.
